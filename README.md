# Java Roadmap & Guide

Welcome to the Java Learning Guide! This repository provides a structured path to master Java from core concepts to advanced microservices and web development, based on the Java Roadmap.

## 🚀 Learning Path

### [1. Java Basics](./docs/01-basics.md)
*   Syntax & Variables
*   Data Types
*   Control Flow (if-else, switch)
*   Loops (for, while, do-while)
*   Arrays (1D, 2D)

### [2. OOPs Concepts](./docs/02-oops.md)
*   Classes & Objects
*   Inheritance (extends, implements)
*   Polymorphism (Overloading, Overriding)
*   Abstraction (Abstract Classes, Interfaces)
*   Encapsulation (Access Modifiers, Getters/Setters)

### [3. Collections Framework](./docs/03-collections.md)
*   List, Set, Map Interfaces
*   Generics
*   Iterators & ListIterators

### [4. Exception Handling](./docs/04-exception-handling.md)
*   Try-Catch-Finally Blocks
*   Throw & Throws Keywords
*   Custom Exceptions

### [5. File I/O](./docs/05-file-io.md)
*   FileReader / FileWriter
*   BufferedReader / BufferedWriter
*   Serialization & Deserialization

### [6. Multithreading & Concurrency](./docs/06-multithreading.md)
*   Thread Class & Runnable Interface
*   Synchronization & Volatile
*   Executors Framework

### [7. Java 8+ Features](./docs/07-java-8-plus.md)
*   Lambda Expressions
*   Stream API
*   Functional Interfaces
*   Date & Time API (java.time)

### [8. JDBC (Database Connectivity)](./docs/08-jdbc.md)
*   Connection Management
*   Statements & PreparedStatements
*   Transaction Management

### [9. Frameworks & Tools](./docs/09-frameworks-and-tools.md)
*   Maven / Gradle (Build Tools)
*   Spring Boot (Microservices)
*   Hibernate (ORM)

### [10. Web Development](./docs/10-web-dev.md)
*   Servlets & JSP
*   REST API Design
*   Spring MVC
